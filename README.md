# 📚 RoomTwenty BookNook

> Full-Stack-Buchverwaltungs-App mit React, Node.js, Express und MongoDB

RoomTwenty BookNook ist eine Full-Stack-Webanwendung für Bücherliebhaber. Die Plattform ermöglicht es Nutzern, Bücher zu entdecken, persönliche Leselisten zu verwalten und ihre Lieblingsbücher an einem zentralen Ort zu organisieren.

Das Projekt wurde als Abschlussprojekt im Rahmen der Full-Stack-Webentwickler-Weiterbildung bei DCI (Digital Career Institute) entwickelt und diente der praktischen Anwendung moderner Webtechnologien im MERN-Stack-Umfeld.

## 🚀 Features

### 📖 Buchverwaltung

- Bücher durchsuchen
- Suche nach Titel, Autor oder Genre
- Detailansicht für einzelne Bücher
- Übersichtliche Darstellung der Buchinformationen

### 👤 Benutzerverwaltung

- Benutzerregistrierung
- Login und Logout
- JWT-basierte Authentifizierung
- Geschützte Benutzerbereiche

### ❤️ Persönliche Bibliothek

- Lieblingsbücher speichern
- Eigene Leselisten erstellen
- Bücher verwalten und organisieren

### 📱 Responsives Design

- Optimiert für Desktop, Tablet und Smartphone
- Benutzerfreundliche und moderne Oberfläche

## ⚙️ Installation

### Repository klonen

```bash
git clone git@github.com:jacqueline-s-w/dci-final-project-roomtwenty-booknook.git
cd dci-final-project-roomtwenty-booknook
```

### Frontend installieren

```bash
cd frontend
npm install
```

### Backend installieren

```bash
cd backend
npm install
```

---

## ▶️ Anwendung lokal starten

### Backend starten

```bash
npm run dev
```

### Frontend starten

```bash
npm run dev
```

Nach dem Start ist die Anwendung über die von Vite bereitgestellte lokale Entwicklungsumgebung erreichbar.
